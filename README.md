# logr
[![GoDoc](https://godoc.org/github.com/go-figure/logr?status.png)](https://godoc.org/github.com/go-figure/logr)

`logr` is a composable, easy to extend logging library.

## Install
```shell
go get github.com/go-figure/logr
```
